# CSC648 Section04 Team04 — Animated Team Site

A ready-to-run React + Vite + Tailwind + React Router + Framer Motion project with two pages:

- **Home**: Team name + animated hero
- **About**: Team information + members grid (animated)

## Quick Start

```bash
npm install
npm run dev
```

Open the local URL shown in terminal (usually http://localhost:5173).

## Customize

- Edit `src/pages/Home.jsx` and `src/pages/About.jsx` to change team info, links, and members.
- Update `Navbar.jsx` to change the title/term.
- Replace placeholder emails/links with real ones.

## Tech

- React 18
- Vite 5
- Tailwind CSS 3
- React Router 6
- Framer Motion 10
- Lucide Icons
